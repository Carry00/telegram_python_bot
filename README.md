# telegram_python_bot
telegram_python_bot机器人，发送文字、图片 （未完善）
### 环境安装
也可以通过pip安装		
```pip install python-telegram-bot ```
或者通过源码安装
````bash
$ git clone https://github.com/python-telegram-bot/python-telegram-bot --recursive
$ cd python-telegram-bot
$ python setup.py install
````
